# SafeStride — ML-Integrated Smart Safety System

An AI/ML-powered personal safety system combining embedded hardware with a mobile app to detect emergencies, trigger alerts, and recommend safer routes in real time.

---

## 🚀 Key Features
- Heel-Tap SOS & Fall Detection via accelerometer
- Real-time GPS location sharing with emergency contacts
- Automated SMS alerts via GSM even offline
- AI-powered safe route recommendations and risk zone prediction
- Safety heatmap and auto-generated incident reports
- Mobile app for device configuration and alert monitoring

---

| Layer | Technology |

| Cloud | Google Cloud Platform |
